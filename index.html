Cash Offers
www.cash-offers.ca

Cash Offers is a single page website built for a real estate entrepreneur. The client wanted a simple single page design with a call to action for converting leads. When viewed on mobile, a supplementary button appears to guide you to the contact form for easier conversions. Along with the ability to contact the owner via telephone through the site, ease of access makes for an easily navigatable website for all users.

Personal Projects

Tic-Tac-Toe React App

This Tic-Tac-Toe app was built using React. Through stateful components this app offers the functionality for users to input their names and keeps track of the score of the game. Once a winner is declared, a small window will pop up with the winners name and an option to play again. The game will continue to keep track of the score until the website/app is reloaded. For more information about this project please check it out on Github.

Subreddit Explorer App

This app was built using the Reddit JSON API. Using React for the framework, the app commuinicates with the Reddit JSON API to view current 25 "Hot" subreddits under the specified search. The posts that are loaded show the image(if there is one), the number of comments, and the user that posted the subreddit. If a user clicks on a post it will direct you to the post on Reddit's website. Please find more information about this project on Github.
